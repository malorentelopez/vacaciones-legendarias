# Avatar illustrations

All character avatar images in this folder are **original AI-generated illustrations** created exclusively for **Verano Legendario**.

- **28 portraits** across 3 themes (Aventura, Manga, Océano)
- Each role adapts visually to gender (boy/girl)
- Family-friendly chibi-anime style, designed for game avatars

## Sync to apps

After replacing images here, run:

```bash
pnpm avatars:sync
```

## Themes

| Tema | Roles |

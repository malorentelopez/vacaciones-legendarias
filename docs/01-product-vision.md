# Product Vision

Verano Legendario es una plataforma de gamificación familiar inspirada en RPG.

Objetivos:
- Crear hábitos.
- Favorecer la autonomía.
- Potenciar creatividad.
- Desarrollar responsabilidad.
- Mantener el verano divertido.

Roles:
- Parent
- Child

Principios:
- Progreso antes que castigo.
- Sistema configurable.
- Diversión.
- Libertad por las tardes.
